"# PPO_lab" 
## Lab 1
1. Проект для Andoid 6.0
2. На данный момент программа должна иметь один экран, на котором должно отображаться версия проекта и IMEI телефона.
## lab 2
1. Провести миграцию проекта. Вместо использования Android Support Library, проект должен использовать AndroidX (https://developer.android.com/jetpack/androidx/)
2. Реализовать на главной странице один из рекомендованных патернов **Navigation drawer** (https://material.io/design/components/navigation-drawer.html) или **Bottom navigation**
3. В созданной системе навигации добавить страницу профиля пользователя со следующей информацией: аватар, имя, фамилия, телефон, email. Должна быть возможность редактирования. При смене аватара должна быть предусмотрена возможность сделать снимок или искользовать фото из галереи.
4. Реализовать возможность перейти на любую страницу с помощью url вида *sdapp://by.myapp/page/N*(N-номер страницы)
5. Добавить AppBar кнопку, при нажатии которой будет открываться страница About, в которую нужно переенести информацию о устройстве и версии программы.
## lab 3
1. Организовать хранение данных пользователя(возможно использование Firebase)
## lab 4
1. Добавить страницу для чтения rss-новостей. При недоступности интернета данные должны браться из кэша.
